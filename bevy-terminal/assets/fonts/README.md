# Fonts Directory

This directory is for font files needed by the terminal plugin.

**Fonts are not included in the repository due to licensing.**

## Download Fonts

Run the download script from the repository root:

```bash
bash scripts/download-fonts.sh
```

This will download Cascadia Mono fonts from Microsoft's official release:
- CascadiaMono-Regular.ttf
- CascadiaMono-Bold.ttf
- CascadiaMono-Italic.ttf
